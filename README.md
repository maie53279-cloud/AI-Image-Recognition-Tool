# AI 图像识别工具

基于 **PyQt5** 和 **百度AI开放平台** 的图像识别桌面应用，支持识别银行卡、身份证、驾驶证、车牌、动植物、Logo等11种类型。

![界面预览](screenshot.png)  <!-- 可替换为实际截图 -->

## ✨ 功能特性

- 🖼️ 支持11种识别类型：银行卡、植物、动物、通用票据、营业执照、身份证、车牌号、驾驶证、行驶证、车型、Logo
- 📋 一键复制识别结果到剪贴板
- 🖱️ 图形化界面，操作简单
- 🔍 图片预览功能
- 🌐 基于百度AI REST API，识别准确率高

## 📋 依赖环境

- Python 3.7+
- PyQt5 >= 5.15.0
- 网络连接（调用百度API）
