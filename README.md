# AI 图像识别工具

基于 **PyQt5** 和 **百度AI开放平台** 的图像识别桌面应用，支持识别银行卡、身份证、驾驶证、车牌、动植物、Logo等11种类型。

![界面预览](screenshot.png)  <!-- 可替换为实际截图 -->

## ✨ 功能特性

- 🖼️ 支持11种识别类型：银行卡、植物、动物、通用票据、营业执照、身份证、车牌号、驾驶证、行驶证、车型、Logo
- 📋 一键复制识别结果到剪贴板
- 🖱️ 图形化界面，操作简单
- 🔍 图片预览功能
- 🌐 基于百度AI REST API，识别准确率高

## 📋 依赖环境

- Python 3.7+
- PyQt5 >= 5.15.0
- 网络连接（调用百度API）

## 快速开始

**第一步：克隆项目到本地**
git clone https://github.com/你的用户名/AI-Image-Recognition-Tool.git
cd AI-Image-Recognition-Tool

**第二步：安装Python依赖**
方式一：使用requirements.txt（推荐）
pip install -r requirements.txt
方式二：手动安装
pip install PyQt5>=5.15.0
验证安装
python -c "import PyQt5; print('PyQt5安装成功')"

**第三步：申请百度AI接口**
3.1 注册/登录百度账号
访问 百度AI开放平台
点击右上角"控制台"
使用百度账号登录（若无账号请先注册）

3.2 创建图像识别应用
登录控制台后，依次点击："产品服务" → "全部产品" → "图像识别"
点击"创建应用"按钮
填写应用信息：应用名称（如"AI图像识别工具"）、应用描述
勾选所需的图像识别接口（建议全选）
点击"立即创建"完成

3.3 获取API密钥
在应用列表中找到刚创建的应用
记录以下两个关键值：API Key 和 Secret Key

⚠️ 请妥善保管这两个密钥，不要泄露给他人或上传到公开仓库。

**第四步：配置API密钥**
打开项目中的 main.py 文件，找到配置区域：

python
# ==================== 配置区域 ====================
API_KEY = "请替换为你的API Key"
SECRET_KEY = "请替换为你的Secret Key"
# =================================================
